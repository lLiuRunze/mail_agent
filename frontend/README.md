# Mail Agent - 前端

智能邮件管理系统的前端界面，基于 React + TypeScript + Vite 构建。

## 功能特性

- 📧 邮件列表展示（收件箱、所有邮件、星标、已发送）
- 🤖 AI 助手聊天界面，支持自然语言操作邮件
- 📝 内联邮件预览和编辑
- 🔍 已读/未读筛选
- 👤 多账号管理
- 🎨 彩色发件人头像（基于哈希算法）

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录

## 技术栈

- React 19
- TypeScript
- Vite
- Axios
- Lucide React (图标)
